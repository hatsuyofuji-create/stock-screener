#業種売買代金（億円）勢い（5日）

J-Quants の実データです。 東証33業種＋分類外「その他」の売買代金（V2の Va）を集計し、毎営業日 GitHub Actions が自動更新して倉庫に保存します。売買代金は日々の振れを抑えるため5営業日移動平均で表示。電気機器は売買代金が突出して大きいため、初期グラフからは外しています（表の行クリックで追加できます）。グラフの縦目盛りは「色つき表示中の業種」に合わせて自動調整します。

投資判断・売買の推奨を行うものではありません（表示専用）。
